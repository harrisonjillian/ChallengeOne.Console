README

Cafe Menu
With this cafe, a meal number, meal name, description, list of ingredients and pricing has been added. This allows the the cafe manager to add, delete and see all the current items at the cafe.  

Claims Department
This application allows for a claims agent to see all claims at one time, take care of the next claim in the queue or go back to the main menu, or enter a new claim.  

Insurance Badges
The badge system for Komodo insurance has been updated to allow a security staff member to add a new badge, add door access to a badge, delete door access from a badge and to show a list with all badge numbers and door access.
